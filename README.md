## Overview

A bash script to validate your yaml files using [yamllint](https://pypi.org/project/yamllint/).

This works on most CICD platforms as well as locally.

## Pre-requisite
 Install [yamllint](https://pypi.org/project/yamllint/) using `pip install yamllint` 

## Basic Usage
1. Save the Script: Save the script to a file, for example, yaml-lint.sh.
2. Make it Executable: Run chmod +x yaml-lint.sh to make the script executable.
3. Run the Script: Execute the script by running ./yaml-lint.sh [directory or file]. If no argument is provided, it defaults to the current directory.
